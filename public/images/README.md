# Images Directory

This directory contains all the images used in the SecurePro security company website.

## Required Images

### Hero Section

- `hero-security.jpg` - Main hero background image showing security professionals or security equipment

### Feature Section

- `security-guard.jpg` - Professional security guard in uniform
- `cctv-monitoring.jpg` - CCTV monitoring station or cameras
- `security-team.jpg` - Group of security professionals
- `mobile-patrol.jpg` - Security vehicle or mobile patrol scene

### Services Section

- `cctv-operations.jpg` - CCTV operations and monitoring
- `door-supervision.jpg` - Door supervisor at work
- `mall-security.jpg` - Mall security operations
- `parking-management.jpg` - Car parking management scene
- `reception-concierge.jpg` - Reception/concierge desk
- `static-patrol.jpg` - Static security guard on duty

### About Section

- `security-team-working.jpg` - Security team working together

## Image Specifications

- **Format**: JPG or PNG
- **Resolution**: High resolution (minimum 1200px width for hero, 800px for others)
- **Aspect Ratio**: 16:9 or 4:3 recommended
- **Quality**: Professional, high-quality images
- **Content**: Security-related, professional, and trustworthy

## Image Sources

You can source images from:

- Unsplash (free)
- Pexels (free)
- Shutterstock (paid)
- iStock (paid)
- Or use your own professional security photos
